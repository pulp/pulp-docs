# Brand Guide

The brand guide is a collection of assets and guidelines for using Pulp's visual identity and related design materials.

It contains, among other things: logo vector images with variations, color information, and supporting materials for printing and presentations.
They live in the Pulp Docs repository in raw format and can be accessed here:
<https://github.com/pulp/pulp-docs/blob/main/brand-assets>
